## 🏃 과정

- 리눅스 커널을 학교 서비스인 JCloud로 생성 후 분석
- CPU 성능 평가를 위해 행렬 곱셈 프로그램 구현
- 1개의 코어로 Context Swithching 성능 분석
- 리눅스 CPU 스케줄러 소스 분석
- LKM을 이용해 스케줄러 구조 분석
- 나만의 CPU 스케줄러 생성 후 FIFO, RR 알고리즘 구현
